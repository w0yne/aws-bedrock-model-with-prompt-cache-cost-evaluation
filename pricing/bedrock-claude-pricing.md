## 缓存定价公式

对所有 Bedrock 上的 Claude 模型：

- **缓存写入** = 输入价格 × 1.25
- **缓存读取** = 输入价格 × 0.10

## 成本计算公式

```
实际成本 = (输入Token × 输入价格 / 1M)
         + (缓存写入Token × 缓存写入价格 / 1M)
         + (缓存读取Token × 缓存读取价格 / 1M)
         + (输出Token × 输出价格 / 1M)
```

## 假设无缓存的成本

```
无缓存成本 = ((输入Token + 缓存读取Token + 缓存写入Token) × 输入价格 / 1M)
           + (输出Token × 输出价格 / 1M)
```

## 节省比例计算

```
节省比例 = (无缓存成本 - 实际成本) / 无缓存成本 × 100%
```
